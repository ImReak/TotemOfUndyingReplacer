# TotemOfUndyingReplacer - 不死图腾替换器

这是一个 **纯客户端 Fabric Mod**，用于在玩家 **生命值低于最大生命值的 一半 时**，自动从背包中查找 **不死图腾** 并 **放入副手**，以降低意外死亡风险。

> 本 Mod 仅通过 **合法的客户端物品栏操作** 实现，不需要服务器安装，对服务器完全无侵入。

---

## ✨ 功能特性

- ❤️ **血量低于 一半 自动装备不死图腾**
- ✋ **副手已持有图腾时不会重复操作**
- 🛡 **防误操作设计**（不在容器界面、不占用鼠标光标）
- 🎮 **行为完全符合原版逻辑**
- 🧩 **无需服务器支持，联机可直接使用**

---

## 🖥 运行环境

- **Minecraft**：1.21.10
- **Mod Loader**：Fabric
- **Java**：>= Java 21

- ❌ 不修改游戏规则或玩家属性

---

## ⚙️ 触发逻辑说明

自动装备不死图腾仅在以下条件 **全部满足** 时触发：

- 玩家生命值 **≤ 最大生命值的 一半**
- 玩家副手 **未持有不死图腾**
- 玩家背包中 **存在不死图腾**
- 当前界面为 **玩家自身背包**（未打开箱子 / 工作台等）
- 鼠标光标 **未持有任何物品**

---

## 🛡 安全性设计

- 若玩家在触发期间手动操作背包：
  - Mod 将会 **放弃本次自动操作**
- **不会丢失物品 / 不会复制物品**

---

## ❓ 常见问题

### Q: 这是作弊 Mod 吗？
A: 不是。它仅在客户端帮玩家完成原本可以手动完成的操作（移动图腾到副手），不修改任何游戏判定。

### Q: 服务器需要安装吗？
A: 不需要。

### Q: 支持联机服务器吗？
A: 支持，只要服务器本身允许不死图腾使用。
